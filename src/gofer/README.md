# Summary of orthorectification
Due to off-nadir scan angles of geostationary satellites and complex terrain, imagery is subject to a parallax effect. Images should undergo orthorectification to improve image correctness.

Super interesting stuff, but I'm definitely not qualified to speak more on the matter.
